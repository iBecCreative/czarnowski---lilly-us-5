# Welcome to Ibex
## What is Ibex?
Ibex is a minimal frontend SCSS framework for creating websites.
**This Style Guide illustrates the scope of what Ibex sets out to do, which is:**
* Provide a quick and easy way for an iBec Developer to generate basic page layouts.
* Make the Typography process easier for multiple devices using fluid type.
* Provide basic styles for the most common elements.
* Get out of your way so you can do great work.

View the full documentation here: https://ibeccreative.github.io/Ibex/
